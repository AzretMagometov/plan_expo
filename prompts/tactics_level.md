# Промпт для создания тактического уровня (OKR или SMART)

## Задача
Создать измеримые цели на 3-12 месяцев, которые ведут к стратегической идентичности.

## Выбор метода

### Используй OKR если:
- Горизонт 3-12 месяцев
- Нужна амбициозность (70% достижения = успех)
- Есть несколько взаимосвязанных результатов
- Работа в команде или продуктовый контекст

### Используй SMART если:
- Горизонт до 3 месяцев
- Нужна конкретная, достижимая цель
- Корпоративный/проектный контекст
- Одна чёткая цель

## Процесс работы с OKR

### Шаг 1: Определение Objective

Objective должен быть:
- Вдохновляющим и качественным (не количественным)
- Связанным со стратегической идентичностью
- Амбициозным, но реалистичным

Задай вопросы:
1. "Какой вдохновляющий результат ты хочешь достичь в ближайшие 3-6 месяцев?"
2. "Как это связано с твоей идентичностью [из стратегического уровня]?"
3. "Что изменится, когда ты достигнешь этого?"

Примеры хороших Objectives:
- ✅ "Стать признанным экспертом в области UX-дизайна"
- ✅ "Получить первую работу junior-разработчиком"
- ✅ "Создать устойчивый поток клиентов для консалтинга"

### Шаг 2: Создание Key Results

Правила для Key Results:
- Только количественные метрики
- 3-5 KR на один Objective
- Амбициозность: 70% достижения = успех
- Измеримость: можно отслеживать прогресс

Задай вопросы:
1. "Какие конкретные метрики покажут, что ты достиг Objective?"
2. "Какой текущий уровень? Какой целевой?"
3. "Как ты будешь измерять прогресс?"

Примеры хороших Key Results:

**Для Objective "Стать экспертом в UX":**
- KR1: Опубликовать 12 статей на Medium с минимум 500 прочтений каждая
- KR2: Выступить на 3 профильных конференциях
- KR3: Набрать 5000 подписчиков в профессиональном блоге
- KR4: Получить сертификацию Google UX Design

**Для Objective "Получить работу junior-разработчиком":**
- KR1: Завершить 3 pet-проекта на GitHub со звёздами
- KR2: Внести contribution в 5 open-source проектов
- KR3: Пройти 20 собеседований, получить минимум 3 оффера
- KR4: Написать 10 технических статей, минимум 2 в топе Хабра

### Шаг 3: Проверка качества OKR

Убедись, что:
- ✅ Objective вдохновляет (не просто метрика)
- ✅ Key Results измеримы (числа, проценты, даты)
- ✅ KR не являются задачами ("провести исследование" — это задача, не KR)
- ✅ Амбициозность на уровне 70% (если 100% достижимо — недостаточно амбициозно)
- ✅ Максимум 3-5 Objectives одновременно

## Процесс работы со SMART

### Шаг 1: Проверка каждого критерия

**S - Specific (Конкретная):**
- Что именно? Где? Когда? С кем?

**M - Measurable (Измеримая):**
- Как измерить? Какие метрики?

**A - Achievable (Достижимая):**
- Реалистична ли цель? Есть ли ресурсы?

**R - Relevant (Релевантная):**
- Связана ли со стратегической идентичностью?

**T - Time-bound (Ограниченная во времени):**
- Конкретная дата или период

### Примеры SMART-целей:

✅ **Правильно:** "Пройти курс по Python до 1 июня, выполнив минимум 80% практических заданий"

✅ **Правильно:** "Снизить операционные расходы на 10% к 31 марта через автоматизацию"

❌ **Неправильно:** "Выучить программирование" (не конкретно, не измеримо)

❌ **Неправильно:** "Оптимизировать бизнес" (размыто, нет метрик)

### Шаг 4: Определение временных рамок

Для OKR:
- Квартальные OKR (3 месяца) — стандарт
- Полугодовые OKR (6 месяцев) — для долгосрочных целей
- Годовые OKR — только для стратегических целей

Для SMART:
- До 3 месяцев — оптимально
- Можно разбить большую цель на несколько SMART-целей

### Типичные ошибки (избегай их):

❌ **KR как задачи** — "провести исследование" это не KR, это задача
❌ **Слишком много Objectives** — максимум 3-5 на квартал
❌ **100% достижимость** — значит недостаточно амбициозно
❌ **Игнорирование R (Relevant)** — цель не связана со стратегией
❌ **Нереалистичные сроки** — лучше заложить буфер 20-30%

### Результат работы

После создания тактического уровня представь результат в формате:

```
┌─────────────────────────────────────┐
│   ТАКТИКА (Что? Как измерить?)     │
│                                     │
│   Objective: [вдохновляющая цель]  │
│                                     │
│   Key Results:                      │
│   - KR1: [метрика]                  │
│   - KR2: [метрика]                  │
│   - KR3: [метрика]                  │
│                                     │
│   Текущий уровень: [значение]      │
│   Целевой уровень: [значение]      │
│   Горизонт: 3-12 месяцев           │
└─────────────────────────────────────┘
```

Или для SMART:

```
┌─────────────────────────────────────┐
│   ТАКТИКА (SMART-цель)              │
│                                     │
│   Цель: [полная формулировка]      │
│                                     │
│   S: [конкретика]                   │
│   M: [метрики]                      │
│   A: [достижимость]                 │
│   R: [релевантность]                │
│   T: [временные рамки]              │
│                                     │
│   Горизонт: до 3 месяцев            │
└─────────────────────────────────────┘
```

Затем скажи: "Отлично! Теперь у нас есть измеримые цели. Переходим к операционному уровню — создадим ежедневные действия и привычки. Готов?"
