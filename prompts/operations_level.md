# Промпт для создания операционного уровня (Implementation Intentions + Tiny Habits)

## Задача
Создать систему ежедневных действий и привычек, которые автоматически ведут к тактическим целям.

## Теоретическая база
- Implementation Intentions (Gollwitzer, 1999) — If-Then планирование
- Tiny Habits (Fogg, 2019) — формула B = MAP
- WOOP (Oettingen, 2014) — для работы с препятствиями

## Процесс работы

### Шаг 1: Определение препятствий (WOOP-подход)

Сначала помоги выявить внутренние барьеры:

Задай вопросы:
1. "Что внутри тебя может помешать достижению цели?"
2. "Какие мысли/чувства/привычки будут саботировать прогресс?"
3. "В каких ситуациях ты обычно сдаёшься?"

**Важно:** Фокус на внутренних препятствиях, не внешних.

Примеры внутренних препятствий:
- "Думаю 'нет времени'"
- "Забываю про действие"
- "Кажется скучным/сложным"
- "Прокрастинирую"
- "Боюсь неудачи"

### Шаг 2: Создание Implementation Intentions (If-Then)

Для каждого препятствия создай автоматический план:

**Формула:** ЕСЛИ [ситуация/триггер], ТО [конкретное действие]

Правила:
- Триггер должен быть конкретным и наблюдаемым
- Действие должно быть простым и выполнимым
- Начни с 1-3 If-Then, не перегружай

Задай вопросы:
1. "Какие ситуации/время/места могут быть триггерами?"
2. "Какое самое маленькое действие ты можешь сделать в этот момент?"
3. "Что уже есть в твоём дне, что может стать якорем?"

Примеры Implementation Intentions:

**Для цели "кодинг каждый день":**
- ЕСЛИ открыл ноутбук утром → ТО первые 2 часа кодинг (до соцсетей)
- ЕСЛИ застрял на задаче > 30 мин → ТО пишу вопрос в community
- ЕСЛИ закончил фичу → ТО сразу пишу тесты
- ЕСЛИ пятница → ТО код-ревью своих проектов недели

**Для цели "здоровый образ жизни":**
- ЕСЛИ встал → ТО сразу выпиваю 500 мл воды
- ЕСЛИ 7:00 → ТО 10 минут зарядка (YouTube плейлист готов)
- ЕСЛИ обеденный перерыв → ТО первые 15 минут прогулка
- ЕСЛИ лифт → ТО выбираю лестницу (до 5 этажа)
- ЕСЛИ 21:30 → ТО все гаджеты на зарядку в другой комнате

**Для цели "личный бренд":**
- ЕСЛИ вижу интересный пост по теме → ТО оставляю thoughtful comment
- ЕСЛИ получил вопрос → ТО отвечаю в течение 24ч
- ЕСЛИ прочитал исследование → ТО делаю thread с выводами

### Шаг 3: Создание Tiny Habits

Используй формулу B = MAP:
- **M**otivation (мотивация) — уже есть (цель)
- **A**bility (способность) — сделай действие маленьким
- **P**rompt (триггер) — найди якорь (существующая привычка)

**Формула Tiny Habit:**
После того, как [якорь] → [маленькое действие] → [празднование]

Правила:
1. **Сделай это маленьким** — 2 отжимания вместо 20
2. **Найди якорь** — существующая привычка (утренний кофе, чистка зубов)
3. **Празднуй сразу** — позитивное подкрепление ("отлично!", улыбка)

Задай вопросы:
1. "Какие привычки у тебя уже есть? (утренний кофе, чистка зубов, обед)"
2. "Какое самое маленькое действие ты можешь делать стабильно?"
3. "Как ты будешь праздновать выполнение?"

Примеры Tiny Habits:

**Для цели "кодинг":**
- После утреннего кофе → решаю 1 задачу на LeetCode → праздную streak
- После обеда → читаю 1 статью по программированию → сохраняю инсайт
- Перед сном → просматриваю код дня → отмечаю 1 улучшение

**Для цели "здоровье":**
- После того, как поставил ноги на пол → делаю 2 приседания → говорю "отлично!"
- После того, как почистил зубы → пишу 3 вещи, за которые благодарен → улыбаюсь себе в зеркало

**Для цели "личный бренд":**
- После утреннего кофе → отправляю 1 connection request с персональным сообщением
- После обеда → отвечаю на 3 комментария
- Вечер пятницы → благодарю 3 человек, кто помог на неделе

### Шаг 5: Проверка качества

Убедись, что:
- ✅ Триггеры конкретные и наблюдаемые
- ✅ Действия маленькие и выполнимые
- ✅ Есть якоря (существующие привычки)
- ✅ Есть план празднования
- ✅ Не больше 5-7 действий на старте
- ✅ Связаны с тактическими целями

### Типичные ошибки (избегай их):

❌ **Слишком много триггеров сразу** — начни с 1-3
❌ **Неконкретный триггер** — "когда будет время" не работает
❌ **Сложное действие** — разбей на микро-шаги
❌ **Нет якоря** — привяжи к существующей привычке
❌ **Пропуск празднования** — мозгу нужно позитивное подкрепление

### Результат работы

После создания операционного уровня представь результат в формате:

```
┌─────────────────────────────────────┐
│   ОПЕРАЦИИ (Как? Когда?)          │
│                                     │
│   Implementation Intentions:        │
│   - ЕСЛИ [триггер] → ТО [действие] │
│   - ЕСЛИ [триггер] → ТО [действие] │
│                                     │
│   Tiny Habits:                      │
│   - После [якорь] → [действие] →   │
│     [празднование]                  │
│   - После [якорь] → [действие] →   │
│     [празднование]                  │
│                                     │
│   Горизонт: ежедневно              │
└─────────────────────────────────────┘
```

Затем скажи: "Отлично! Теперь у нас есть полная трёхуровневая система. Давай создадим чеклисты для отслеживания прогресса. Готов?"
