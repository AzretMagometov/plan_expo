# Changelog

## [Unreleased] - 2025-12-21

### Добавлено

#### Система отслеживания изменений целей

- **Новый промпт `goal_change_tracking.md`** для классификации изменений целей
- **Типы изменений:**
  - `PROGRESS` - естественный прогресс, обновление метрик
  - `FORCED_CHANGE` - вынужденные изменения (авария, кризис, форс-мажор)
  - `VOLUNTARY_CHANGE` - добровольные изменения (переосмысление, новые приоритеты)
  - `PLAN_ADJUSTMENT` - корректировка тактических/операционных планов

- **Расширенная структура файла цели:**
  - Поле "Подстатус" для paused/cancelled целей
  - Поле "Тип последнего изменения"
  - Секция "КРИТИЧЕСКИЕ СОБЫТИЯ" для вынужденных изменений
  - Улучшенная "ИСТОРИЯ ИЗМЕНЕНИЙ" с типами изменений

- **Автоматическое выявление критических событий:**
  - Скрипт `analyze_reflection.py` теперь выявляет критические события в рефлексиях
  - Поиск ключевых слов для вынужденных и добровольных изменений
  - Автоматические рекомендации по обновлению целей

### Изменено

- Обновлён `prompts/goal_management.md` с новой структурой
- Обновлён `scripts/analyze_reflection.py` для выявления критических событий
- Обновлён `.cursorrules` с правилами работы с изменениями целей

### Решённые проблемы

- Система теперь может различать вынужденные и добровольные изменения целей
- Критические события документируются в отдельной секции
- Не нужно ковыряться в рефлексиях - вся информация о критических изменениях в файле цели
