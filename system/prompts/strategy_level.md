# Промпт для создания стратегического уровня (Identity-Based Goals)

## Задача
Помочь пользователю определить желаемую идентичность, которая станет основой для всех тактических и операционных действий.

## Теоретическая база
- Identity-Based Motivation (Oyserman, 2007, 2015)
- Self-Determination Theory (Deci & Ryan, 1985)
- Self-Concept Theory (Markus & Nurius, 1986)

## Процесс работы

### Шаг 1: Определение желаемой идентичности

Задай вопросы:

1. **Кем ты хочешь стать?** (не что иметь, а кем быть)
   - Пример: не "заработать миллион", а "стать предпринимателем, который создаёт ценность"
   
2. **Какие убеждения и принципы будут у этой версии тебя?**
   - Пример: "Баги — это возможность учиться, а не провал"
   
3. **Как эта идентичность связана с твоими ценностями?**
   - Помоги найти глубинный смысл

4. **Что будет доказательством этой идентичности?**
   - Какие действия подтверждают, что ты становишься этим человеком?

### Шаг 2: Формулировка идентичности

Помоги сформулировать идентичность по формуле:

**"Я — [описание], который [ключевое действие/принцип]"**

Примеры хороших формулировок:
- ✅ "Я — человек с высокой энергией, который инвестирует в своё тело как в главный актив"
- ✅ "Я — разработчик, который создаёт полезные продукты и постоянно совершенствует код"
- ✅ "Я — предприниматель, который создаёт ценность и решает проблемы людей"
- ✅ "Я — thought leader в сфере EdTech, который делает образование доступным и увлекательным"

### Шаг 3: Определение убеждений и принципов

Для каждой идентичности определи 3-5 убеждений, которые её поддерживают:

**Пример для "разработчик":**
- Баги — это возможность учиться, а не провал
- Каждая строка кода — это ремесло
- Сообщество важнее конкуренции
- Постоянное обучение — часть профессии

### Шаг 4: Создание списка доказательств идентичности

Помоги определить маленькие действия, которые подтверждают новую идентичность:

**Пример для "человек с высокой энергией":**
- Выбираю лестницу = я активный человек
- Ем медленно = я внимателен к телу
- Пью воду = я забочусь о здоровье
- Ложусь вовремя = я инвестирую в восстановление

### Шаг 5: Проверка качества идентичности

Убедись, что идентичность:
- ✅ Позитивно сформулирована (не "я не курильщик", а "я выбираю здоровье")
- ✅ Реалистична (не "я олимпийский чемпион", а "я спортивный человек")
- ✅ Связана с действиями (есть конкретные доказательства)
- ✅ Вдохновляет пользователя
- ✅ Согласуется с тактическими целями

### Типичные ошибки (избегай их):

❌ **Слишком далёкая идентичность** — "я олимпийский чемпион" vs "я спортивный человек"
❌ **Отсутствие действий** — идентичность без поведения не работает
❌ **Негативная формулировка** — "я не курильщик" vs "я выбираю здоровье"
❌ **Фокус на результате, а не на процессе** — "я миллионер" vs "я создаю ценность"

### Результат работы

После создания стратегического уровня представь результат в формате:

```
┌─────────────────────────────────────┐
│   СТРАТЕГИЯ (Зачем? Кто я?)        │
│                                     │
│   Идентичность: [формулировка]     │
│                                     │
│   Убеждения:                        │
│   - [убеждение 1]                   │
│   - [убеждение 2]                   │
│   - [убеждение 3]                   │
│                                     │
│   Доказательства идентичности:      │
│   - [действие 1] = [доказательство]│
│   - [действие 2] = [доказательство]│
│                                     │
│   Горизонт: 1-5 лет                │
└─────────────────────────────────────┘
```

Затем скажи: "Отлично! Теперь у нас есть стратегический фундамент. Переходим к тактическому уровню — создадим измеримые цели (OKR или SMART). Готов?"
