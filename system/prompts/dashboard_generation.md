# Промпт для генерации дашбордов

## Задача
Создание визуализаций прогресса, графиков и отчётов на основе данных из целей и рефлексий.

## Структура директории dashboards/

```
dashboards/
├── daily/
│   └── 2024-01-15.md
├── weekly/
│   └── week_01_2024.md
├── monthly/
│   └── 2024-01.md
├── quarterly/
│   └── Q1_2024.md
└── yearly/
    └── 2024.md
```

## Типы дашбордов

### 1. Ежедневный дашборд

**Содержание:**
- Выполнение операций за день (%)
- Выполнение тактических задач (%)
- Доказательства идентичности (количество)
- Streak (дни подряд)
- График выполнения привычек за неделю

**Формат:**
```markdown
# Дашборд: [дата]

## Выполнение операций

**Общий процент:** [%]

### Implementation Intentions:
- [действие]: [выполнено/пропущено] ([%] за неделю)
- [действие]: [выполнено/пропущено] ([%] за неделю)

### Tiny Habits:
- [привычка]: [выполнено/пропущено] (Streak: [дни])
- [привычка]: [выполнено/пропущено] (Streak: [дни])

## Прогресс по тактике

[Для каждой активной цели:]

### [Название цели]

**Objective/SMART:** [формулировка]

**Key Results:**
- KR1: [текущее]/[целевое] ([%])
- KR2: [текущее]/[целевое] ([%])

**Изменение за день:** [+/-значение]

## Доказательства идентичности

**Всего накоплено:** [число]
**За сегодня:** [число]

[Список последних доказательств]

## Тренды

[График выполнения за последние 7 дней]
[График прогресса по метрикам]
```

### 2. Еженедельный дашборд

**Содержание:**
- Сводка по всем целям
- Тренды выполнения за неделю
- Сравнение с предыдущей неделей
- Топ достижений
- Области для улучшения

### 3. Ежемесячный дашборд

**Содержание:**
- Прогресс по всем тактическим целям
- Тренды за месяц
- Достижения и прорывы
- Анализ препятствий
- Прогноз на следующий месяц

### 4. Квартальный дашборд

**Содержание:**
- Оценка достижения OKR (70% = успех)
- Прогресс к стратегической идентичности
- Крупные изменения и адаптации
- Сравнение с началом квартала
- Планирование следующего квартала

### 5. Годовой дашборд

**Содержание:**
- Достижение стратегических целей
- Трансформация идентичности
- Крупные достижения за год
- Уроки и инсайты
- Видение на следующий год

## Процесс генерации дашборда

### Шаг 1: Сбор данных

1. **Прочитай все активные цели** из `goals/`
2. **Прочитай рефлексии** за нужный период:
   - Для ежедневного: рефлексия за день
   - Для еженедельного: все рефлексии за неделю
   - Для ежемесячного: все рефлексии за месяц
   - И т.д.
3. **Извлеки метрики:**
   - Проценты выполнения
   - Текущие значения метрик
   - Количество доказательств идентичности
   - Streak и тренды

### Шаг 2: Расчёт показателей

1. **Выполнение операций:**
   - Процент выполненных Implementation Intentions
   - Процент выполненных Tiny Habits
   - Общий процент

2. **Прогресс по тактике:**
   - Текущие значения метрик
   - Процент прогресса к цели
   - Изменение за период

3. **Доказательства идентичности:**
   - Всего накоплено
   - За период
   - Тренд

4. **Streak:**
   - Дни подряд выполнения привычек
   - Максимальный streak
   - Текущий streak

### Шаг 3: Создание визуализаций

Используй текстовые графики (ASCII) или описание для:
- Графиков выполнения за период
- Трендов прогресса
- Сравнений с предыдущими периодами

Пример текстового графика:
```
Выполнение операций (последние 7 дней):
Пн: ████████░░ 80%
Вт: ██████████ 100%
Ср: ██████░░░░ 60%
Чт: █████████░ 90%
Пт: ████████░░ 80%
Сб: █████░░░░░ 50%
Вс: ██████████ 100%
```

### Шаг 4: Генерация инсайтов

1. **Выяви паттерны:**
   - Что работает хорошо
   - Что не работает
   - Когда лучше выполнение
   - Какие препятствия частые

2. **Создай рекомендации:**
   - Что продолжать делать
   - Что изменить
   - Что добавить

3. **Предложи адаптации:**
   - Если метрики нереалистичны
   - Если действия слишком сложные
   - Если нужны новые привычки

### Шаг 5: Сохранение дашборда

1. Создай файл в соответствующей директории
2. Заполни структуру данными
3. Добавь визуализации
4. Включи инсайты и рекомендации

## Формат файла дашборда

```markdown
# Дашборд: [тип] - [период]

**Дата генерации:** [дата]
**Период:** [дата начала] - [дата окончания]

## Общая статистика

[Ключевые показатели]

## Прогресс по целям

[Детализация по каждой цели]

## Визуализации

[Графики и диаграммы]

## Инсайты

[Анализ и паттерны]

## Рекомендации

[Что делать дальше]
```

## Интеграция с другими компонентами

- **Цели:** Используй данные из `goals/` для метрик
- **Рефлексии:** Анализируй `reflections/` для трендов
- **Обновление целей:** Если дашборд показывает проблемы, предложи обновить цели
