# ИИ-система целеполагания

Система для помощи пользователям в формулировке целей, создании трёхуровневой структуры достижения, генерации чеклистов и отслеживании прогресса.

## Структура проекта

```
plan_expo/
├── goal_setting_guide.md          # Научно обоснованный гайд по целеполаганию
├── .cursorrules                   # Правила работы ИИ-системы
├── README.md                      # Этот файл
├── DATA_STRUCTURE.md              # Документация по структуре данных
├── goals/                         # Директория для целей пользователя
├── reflections/                   # Директория для рефлексий (daily, weekly, monthly, etc.)
├── dashboards/                    # Директория для дашбордов и визуализаций
├── scripts/                       # Python скрипты
│   ├── generate_reflection.py     # Генерация шаблонов рефлексий
│   └── analyze_reflection.py      # Анализ заполненных рефлексий
└── prompts/                       # Промпты для ИИ-коуча
    ├── system_prompt.md           # Системный промпт для ИИ-коуча
    ├── goal_formulation.md        # Промпт для формулировки цели
    ├── strategy_level.md          # Промпт для стратегического уровня
    ├── tactics_level.md           # Промпт для тактического уровня
    ├── operations_level.md        # Промпт для операционного уровня
    ├── checklist_creation.md      # Промпт для создания чеклистов
    ├── progress_tracking.md       # Промпт для отслеживания прогресса
    ├── goal_management.md          # Управление целями (сохранение, обновление)
    ├── reflection_management.md   # Управление рефлексиями
    └── dashboard_generation.md     # Генерация дашбордов
```

## Как использовать систему

### Для пользователя

1. **Начни диалог с ИИ-коучем:**
   - Скажи: "Я хочу достичь [цель]"
   - Или: "Помоги мне сформулировать цель"

2. **Следуй процессу:**
   - ИИ поможет сформулировать цель
   - Создаст трёхуровневую систему (Стратегия → Тактика → Операции)
   - Сгенерирует чеклисты
   - Поможет отслеживать прогресс

3. **Регулярно обновляй прогресс:**
   - Ежедневно: генерируй рефлексию (`python scripts/generate_reflection.py`), заполняй её, анализируй (`python scripts/analyze_reflection.py`)
   - Еженедельно обновляй метрики в файлах целей
   - Ежемесячно проводи обзор всей системы
   - По запросу создавай дашборды для визуализации прогресса

### Для разработчика/настройки ИИ

1. **Изучи гайд:**
   - Прочитай `goal_setting_guide.md` для понимания методологии

2. **Используй промпты:**
   - Каждый промпт в `prompts/` содержит инструкции для конкретного этапа
   - Используй их последовательно при работе с пользователем

3. **Следуй правилам:**
   - `.cursorrules` содержит все правила работы системы
   - Соблюдай трёхуровневую структуру
   - Используй только научно обоснованные методы

## Трёхуровневая система

### Уровень 1: СТРАТЕГИЯ (Identity-Based Goals)
- **Горизонт:** 1-5 лет
- **Фокус:** Желаемая идентичность, убеждения, принципы
- **Метод:** Identity-Based Motivation

### Уровень 2: ТАКТИКА (OKR или SMART)
- **Горизонт:** 3-12 месяцев
- **Фокус:** Измеримые цели, метрики прогресса
- **Методы:** OKR (для среднесрочных) или SMART (для краткосрочных)

### Уровень 3: ОПЕРАЦИИ (Implementation Intentions + Tiny Habits)
- **Горизонт:** Ежедневно
- **Фокус:** Конкретные действия, триггеры, привычки
- **Методы:** If-Then планирование, Tiny Habits

## Научная база

Система основана на следующих исследованиях:

- **SMART Goals:** Doran (1981), Locke & Latham (2002)
- **WOOP:** Oettingen (2014), Duckworth et al. (2013)
- **Implementation Intentions:** Gollwitzer (1999), Gollwitzer & Sheeran (2006)
- **Identity-Based Motivation:** Oyserman (2007, 2015)
- **Self-Determination Theory:** Deci & Ryan (1985, 2000)
- **Tiny Habits:** Fogg (2019), Lally et al. (2010)

Все методы описаны в `goal_setting_guide.md` с примерами и научными обоснованиями.

## Примеры использования

### Пример 1: Переход в IT

**Стратегия:** "Я — разработчик, который создаёт полезные продукты"

**Тактика (OKR):**
- Objective: Получить первую работу junior-разработчиком
- KR1: Завершить 3 pet-проекта на GitHub
- KR2: Внести contribution в 5 open-source проектов
- KR3: Пройти 20 собеседований, получить 3 оффера

**Операции:**
- ЕСЛИ открыл ноутбук утром → ТО первые 2 часа кодинг
- После утреннего кофе → решаю 1 задачу на LeetCode

### Пример 2: Здоровый образ жизни

**Стратегия:** "Я — человек с высокой энергией, который инвестирует в своё тело"

**Тактика (SMART):**
- Спать минимум 7 часов, ложиться до 23:00, 6 дней в неделю
- 10,000 шагов в день + 3 силовые тренировки в неделю

**Операции:**
- ЕСЛИ встал → ТО сразу выпиваю 500 мл воды
- ЕСЛИ лифт → ТО выбираю лестницу

## Чеклисты

Система создаёт три типа чеклистов:

1. **Identity Evidence List** — доказательства новой идентичности
2. **OKR/SMART Tracker** — отслеживание метрик
3. **Daily Habits Tracker** — ежедневные действия

## Работа с данными

### Хранение целей

Все цели сохраняются в `goals/[goal_id].md`. Каждая цель содержит:
- Стратегический уровень (идентичность, убеждения)
- Тактический уровень (OKR или SMART)
- Операционный уровень (Implementation Intentions, Tiny Habits)
- Историю изменений

### Рефлексии

Система поддерживает рефлексии разных типов:
- **Ежедневные** — в `reflections/daily/YYYY/MM/YYYY-MM-DD.md`
- **Еженедельные** — в `reflections/weekly/YYYY/week_NN_YYYY.md`
- **Ежемесячные** — в `reflections/monthly/YYYY/YYYY-MM.md`
- **Квартальные** — в `reflections/quarterly/YYYY/QN_YYYY.md`
- **Годовые** — в `reflections/yearly/YYYY.md`

### Python скрипты

**Генерация рефлексии:**
```bash
# На сегодня
python scripts/generate_reflection.py

# На конкретную дату
python scripts/generate_reflection.py --date 2024-01-15
```

**Анализ рефлексии:**
```bash
# За сегодня
python scripts/analyze_reflection.py

# За конкретную дату
python scripts/analyze_reflection.py --date 2024-01-15
```

Скрипты автоматически:
- Читают активные цели из `goals/`
- Генерируют шаблоны с планом на день
- Анализируют заполненные рефлексии
- Генерируют комментарии ИИ-системы с рекомендациями

### Дашборды

Дашборды создаются по запросу и содержат:
- Статистику выполнения
- Графики прогресса
- Тренды
- Инсайты и рекомендации

Подробнее см. `DATA_STRUCTURE.md`

## Отслеживание прогресса

- Корректировать планы при необходимости
- Поддерживать мотивацию
- Генерировать рефлексии и дашборды
